# Inverted-Search
Inverted Search system implemented in C that indexes words from multiple files and allows fast word searching using hashing and linked lists. The program creates a database of words with file names and word frequency, enabling efficient document retrieval through command line.

## 📌 Project Overview

• The Inverted Search project is a command-line application developed in C to search words across multiple text files.

• The program reads input files, creates an indexed database of words, and stores the file names along with the frequency of occurrence.

• This allows users to quickly identify the files in which a particular word appears.

• The concept of inverted indexing is widely used in search engines and document retrieval systems.

---

## ✨ Features

• Create database from multiple files  
• Index words using hashing technique  
• Store file names and word frequency  
• Search words quickly from database  
• Display indexed word information
